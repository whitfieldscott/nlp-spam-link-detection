# 🧠 NLP Spam Link Detection

## 📌 Project Overview
This project builds a Natural Language Processing (NLP) model to detect spam links in text messages. It uses text preprocessing and machine learning techniques to classify whether a message is spam or not.

---

## 🎯 Objective
To develop a classification model that can identify spam messages by analyzing textual patterns and word importance.

---

## 🧠 Key Concepts
- Text preprocessing (cleaning, tokenization)
- Stopword removal
- Lemmatization
- TF-IDF vectorization
- Supervised classification

---

## ⚙️ Models Used
- Naive Bayes  
- Support Vector Machine (SVM)

---

## 🔍 Workflow
1. Load dataset  
2. Clean and preprocess text  
3. Remove stopwords and apply lemmatization  
4. Convert text into numerical features using TF-IDF  
5. Train classification models  
6. Evaluate performance  

---

## 📊 Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  

---

## 🛠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- NLTK  

---


---

## 📈 Results
The model successfully detects spam messages by identifying patterns in text and analyzing word importance through vectorization techniques.
